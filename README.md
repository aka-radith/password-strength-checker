# Password Strength Checker

A simple and fun password strength checker built with HTML, CSS, and JavaScript.

This website checks how strong a password is and shows an estimated time to crack it. It also displays useful details like password length, charset size, entropy, requirement checks, and serious advice. To make it more fun, the website also gives sassy feedback based on the password strength.

## Features

- Password strength bar
- Estimated time to crack the password
- Length, charset, and entropy display
- Requirement checklist
- Soft green/red requirement status boxes
- Sassy password comments
- Serious password safety advice
- Show/Hide password button
- Responsive design for mobile and desktop

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Open the website.
2. Type a password in the input box.
3. The website will show:
   - Password strength
   - Estimated crack time
   - Length
   - Charset
   - Entropy
   - Requirement status
   - Feedback and advice

## Important Note

This project is for learning and testing only. Do not enter real bank passwords or important personal passwords on random websites.

## Project Purpose

This project was created to practise front-end web development using HTML, CSS, and JavaScript. It also demonstrates basic password strength checking using simple logic and visual feedback.
